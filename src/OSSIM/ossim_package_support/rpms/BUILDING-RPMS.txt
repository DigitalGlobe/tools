Typical rpm build usage example:

// Build latest trunk code:

// Assumption there a file:
$ ~/rpmbuild/SOURCES/ossim-1.8.19.tar.gz

$ cd ~/rpmbuild/SPECS
$ rpmbuild -ba --define 'RPM_OSSIM_VERSION 1.8.19' --define 'BUILD_RELEASE 30' ossim-trunk-el7.spec

// Example Fedora output:
$  ~/rpmbuild/RPMS/x86_64/ossim-1.8.19-30.fc21.x86_64.rpm
$  ~/rpmbuild/SRPMS/ossim-1.8.19-30.fc21.x86_64.src.rpm
